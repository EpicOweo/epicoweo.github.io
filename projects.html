34-pin 3.5" Floppy to USB

Status: in progress

Repository: https://github.com/EpicOweo/floppy35-to-usb

Overview: The goal of this project is to allow a modern desktop computer to interface with a 3.5" floppy drive. This is accomplished using an Arduino with a custom PCB. The Arduino uses Adafruit's floppy software and shift registers to handle communication to and from the drive, and the host PC communicates with the Arduino via serial.

Current progress: So far, I am working on the hardware implementation. I'm writing code to allow I/O between PISO/SIPO shift registers and the Arduino to interface with the drive. I still need to implement proper serial communication with a computer (currently testing on linux).
